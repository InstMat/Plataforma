Conjunto de los Números Reales

  • Un conjunto es una colección de objetos.

  • Un objeto de un conjunto se conoce como elemento de ese conjunto. Si \(x\) es un elemento de un conjunto \(A\), se denota, \(x\in A\).

  • Se dice que un conjunto \(A\) es un subconjunto de otro conjunto \(B\), si todos los elemento del conjunto \(A\) también son elementos del conjunto \(B\).

  • Si \(A\) es un subconjunto de \(B\) se denota por \(A\subset B\).

Conjuntos numéricos

En este curso, nos centraremos principalmente en los conjuntos numéricos. En particular:

El conjunto de los números naturales se puede expresar como:

\(\mathbb{N} = \{ 1,\, 2, \, 3, \, \ldots \}\)

El conjunto de los números enteros se puede expresar como:

\(\mathbb{Z} = \{ \ldots \, -3, \, -2, \, -1, \, 0,\, 1,\, 2,\, 3,\, \ldots \}\)

Algunos subconjuntos importantes de los enteros:

  • Números enteros positivos: \(\mathbb{Z}^+\)

  • Números enteros negativos: \(\mathbb{Z}^-\)

  • Números pares.

  • Números impares.

  • Números primos.

Conjuntos numéricos

El conjunto de los números racionales se puede definir como:

\[\displaystyle \mathbb{Q} = \left \{ \, \left . \frac{a}{b} \, \right | \, a, b \in \mathbb{Z} , \, b\neq 0 \, \right \}\]

Todo número racional se puede expresar como un decimal finito o decimal infinito periódico.

Conjuntos numéricos

Los números racionales no son suficiente para resolver algunos problemas algebraicos, como por ejemplo:

\[x^2 = 2\] Un número que satisface esa ecuación es \(\sqrt{2}\), raíz cuadrada de 2. El cual no es un número racional.

Al conjunto de números que no pueden escribirse como una división de enteros, se llama conjunto de los números irracionales. El cual se denota por \(\mathbb{I}\) o \(\mathbb{Q}^c\).

Conjunto de los Números Reales

El conjunto de los números reales, \(\mathbb{R}\), es la unión del conjunto de los números racionales, (por lo tanto contiene a los números naturales y enteros), y de los números irracionales. Es decir: \[\mathbb{R}=\mathbb{Q} \cup \mathbb{Q}^c.\]

Conjunto de los Números Reales

Para \(x\), \(a\), \(b\), \(c\), y \(d\) números reales se cumple:

  • Se suponen conocidas las propiedades: conmutatividad, asociatividad, distributividad, existencia de neutros y existencia de inversos de las operaciones adición y multiplicación en los números reales.

  • Si \(a+b=a+c\), entonces \(b=c\).

  • Si \(ab=ac\),     \(a\neq 0\), entonces \(b=c\).

  • Posibilidad de sustracción: Si \(a + x = b\), entonces \(x = b - a\).

  • Posibilidad de división: Si \(ax = b\),   \(a\neq 0\), entonces \(x = b/a\).

  • Si \(ab=0\), entonces \(a=0\)   o   \(b=0\).

  • \(\displaystyle \frac{a}{b} \cdot \frac{c}{d}=\frac{a\cdot c}{b\cdot d}\), \(b\neq 0\), \(d\neq 0\).

  • \(\displaystyle \frac{a}{b} \div \frac{c}{d}=\frac{a\cdot d}{b\cdot c}\), \(b\neq 0\), \(c\neq 0\), \(d\neq 0\).

  • \(\displaystyle \frac{a}{b} \pm \frac{c}{d}=\frac{ad\pm bc}{b d}\), \(b\neq 0\), \(d\neq 0\).

Recta Real

Los números reales pueden ser representados por puntos sobre una recta. Se escoge un punto de referencia, llamado origen, que corresponde al \(0\).

  • Los números reales son ordenados.

  • Decimos que “\(a\) es menor que \(b\)” y escribimos \(a< b\), si \(b-a\) es un número positivo.

  • Geométricamente, esto significa que \(a\) está a la izquierda de \(b\) en la recta numérica,

  • También podemos decir que “\(b\) es mayor que \(a\)” y escribimos \(b>a\).

Recta Real

  • El símbolo \(a \leq b\) quiere decir que \(a<b\) o que \(a=b\) y se lee “\(a\) es menor o igual a \(b\)”.

  • El símbolo \(a \geq b\) quiere decir que \(a>b\) o que \(a=b\) y se lee “\(a\) es mayor o igual a \(b\)”.

Propiedades de los números reales

Sean \(a\), \(b\) y \(c \in \mathbb{R}\). Entonces:

  1. \(\displaystyle a^2 \geq 0\).

  2. \(\displaystyle a\leq b\) si y sólo si \(a \pm c \leq b \pm c\).

  3. Si \(\displaystyle c>0\) tenemos \(a\leq b\) si y sólo si \(a\cdot c \leq b \cdot c\).

  4. Si \(\displaystyle c<0\) tenemos \(a\leq b\) si y sólo si \(a\cdot c \geq b \cdot c\).

  5. \(\displaystyle a\cdot b \geq 0\) si y sólo si \(( a \geq 0\) y \(b \geq 0)\) o \((a \leq 0\) y \(b \leq 0 )\).

  6. \(\displaystyle a\cdot b \leq 0\) si y sólo si \(( a \geq 0\) y \(b \leq 0)\) o \((a \leq 0\) y \(b \geq 0 )\).

Intervalos

Sean \(a\), \(b\) números reales tales que \(a<b\). Los siguientes subconjuntos de \(\mathbb{R}\) son llamados Intervalos.

  1. \(\displaystyle [a,b] = \{ x \in \mathbb{R} \, : \, a \leq x \leq b \} \qquad\)

  2. \(\displaystyle ]a,b[ = \{ x \in \mathbb{R} \, : \, a < x < b \} \qquad\)

  3. \(\displaystyle [a,b[ = \{ x \in \mathbb{R} \, : \, a \leq x < b \} \qquad\)

  4. \(\displaystyle ]a,b] = \{ x \in \mathbb{R} \, : \, a < x \leq b \} \qquad\)

Intervalos

  1. \(\displaystyle [a,+\infty [ = \{ x \in \mathbb{R} \, : \, x \geq a \} \qquad\)

  2. \(\displaystyle ]a,+\infty[ = \{ x \in \mathbb{R} \, : \, x > a \} \qquad\)

  3. \(\displaystyle ]-\infty ,b] = \{ x \in \mathbb{R} \, : \, x\leq b \} \qquad\)

  4. \(\displaystyle ]-\infty , b[ = \{ x \in \mathbb{R} \, : \, x < b \} \qquad\)
