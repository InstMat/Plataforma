Recuerdo de derivadas vistas en la clase pasada

  1. Si \(f(x) = a\) donde \(a\) es una constante, entonces \(f'(x) = 0\)

  2. Si \(f(x) = x^n\), y \(n\) es un número racional, entonces \(f'(x) = nx^{n-1}\)

  3. Si \(f(x) = \sqrt{x}\), entonces \(\displaystyle f'(x) = \frac{1}{2\sqrt{x}}\)

  4. Si \(f(x) = e^x\) , entonces \(f'(x) = e^x\)

  5. Si \(f(x) = \ln x\) , entonces \(f'(x) = \dfrac{1}{x}\)

Derivadas de funciones

Dada la función \(y = f(x)\) entonces la derivada de \(f\) se puede escribir de las siguientes maneras \[y', \quad f'(x) , \quad \frac{dy}{dx} , \quad \frac{df}{dx}.\]

En el caso en que escribamos \(\dfrac{dy}{dx}\) o \(\dfrac{df}{dx}\) diremos que estamos usando la notación de Liebniz.

Sean \(f(x)\) y \(g(x)\) funciones derivables. Entonces

  1. \(\displaystyle (k\cdot f(x))' = k \cdot f'(x)\) donde \(k\) es una constante

  2. \(\displaystyle (f(x) + g(x))' = f'(x) + g'(x)\)

  3. \(\displaystyle (f(x) - g(x))' = f'(x) - g'(x)\)

  4. \(\displaystyle (f(x) \cdot g(x))' = f'(x) \cdot g(x) + f(x) \cdot g'(x)\)

  5. \(\displaystyle \left ( \frac{f(x)}{g(x)} \right )' = \frac{f'(x) \cdot g(x) - f(x) \cdot g'(x)}{(g(x))^2}\)

Ejemplos

Encontrar las derivadas de las siguientes funciones

  1. \(\displaystyle f(x) = 2x^5 + 4x^3 - x +5\)

  2. \(\displaystyle g(x) = \frac{x^2 +2x}{\sqrt{x}}\)

  3. \(\displaystyle h(x) = x e^x\)

  4. \(\displaystyle l(x) = (2x+1)(x^2+6)(3x^5-4)\)

  5. \(\displaystyle p(x) = \frac{x^2+x-3}{x^3+4}\)

Derivadas de orden superior

Si \(f\) es una función derivable, entonces su derivada \(f'\) también es una función, así que \(f'\) puede tener una derivada de sí misma, denotada por \((f')' = f''\). Esta nueva función \(f''\) se llama segunda derivada de \(f\) porque es la derivada de la derivada de \(f\).

Utilizando la notación de Leibniz, la segunda derivada de \(y=f(x)\) se escribe como \[\frac{d}{dx} \left ( \frac{dy}{dx} \right ) = \frac{d^2 y}{dx^2}\]

Encontrar la segunda derivada de las siguientes funciones

  1. \(f(x) = x^4+2x^3-x^2+3x+7\)

  2. \(g(x) = \ln x\cdot \ln x=\left(\ln x\right)^2\)

  3. \(h(x) = x^2 e^x\)

Derivadas de orden superior

De manera similar la derivada de la segunda derivada de \(f\), se llama tercera derivada y se denota por \(f'''\) y así sucesivamente.

Notación:

\(\bullet\) Primera derivada \(y'\) \(f'\) \(\displaystyle \frac{dy}{dx}\)
\(\bullet\) Segunda derivada \(y''\) \(f''\) \(\displaystyle \frac{d^2y}{dx^2}\)
\(\bullet\) Tercera derivada \(y'''\) \(f'''\) \(\displaystyle \frac{d^3y}{dx^3}\)
\(\bullet\) Cuarta derivada \(y^{(iv)}\) \(f^{(iv)}\) \(\displaystyle \frac{d^4y}{dx^4}\)
\(\vdots\) \(\vdots\)
\(\bullet\) \(n\)-ésima derivada \(y^{(n)}\) \(f^{(n)}\) \(\displaystyle \frac{d^n y}{dx^n}\)

Ejemplos

Determinar la primera, segunda y tercera derivada de las siguientes funciones.

    1. \(f(x) = 3x^2+x-4\)
    2. \(f(x) = e^{2x}\)
    3. \(f(x) = x\cdot \ln (x)\)
    4. \(f(x) = (2x^2+1)^2\)

  2. Probar que la función \(\displaystyle y = x\cdot \ln (x)\) satisface la siguientes ecuación \[x^2\cdot y''-x\cdot y' +y =0.\]
