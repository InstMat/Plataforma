Separación de variables

Este método aplica para ecuaciones de la forma \[y'=f(x)g(y),\] donde \(f(x)\) y \(g(y)\) son funciones conocidas. Para resolver este tipo de ecuaciones la idea es “despejar” variables: \[\begin{aligned} y'=f(x)g(y) \Leftrightarrow \frac{dy}{dx}=f(x)g(y) \Rightarrow \frac{1}{g(y)}\mathrm dy=f(x)\mathrm dx\\ \end{aligned}\] de donde podemos integrar para obtener \[\int \frac{1}{g(y)}\mathrm dy=\int f(x)\mathrm dx\] Luego si denotamos \(G(y)=\int \frac{1}{g(y)}dy\) y \(F(x)=\int f(x)dx\) a las respectivas primitivas, obtenemos la solución implícita \[G(y)=F(x)+C.\]
