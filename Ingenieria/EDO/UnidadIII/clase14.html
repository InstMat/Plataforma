Ecuaciones diferenciales ordinarias

Ingeniería Civil

Transformada de Laplace para resolver sistemas de EDOs lineales

Un sistema de ecuaciones diferenciales lineales de primer orden consiste de \(n\) funciones incógnitas \(x_1(t), x_2(t), \ldots, x_n(t)\) y \(n\) ecuaciones diferenciales de la forma

\[ \left\{\begin{aligned} \dfrac{dx_1}{dt}&=a_{11}(t)x_1(t)+a_{12}(t)x_2(t)+\ldots+a_{1n}(t)x_n(t)+f_1(t)\\ \dfrac{dx_2}{dt}&=a_{21}(t)x_1(t)+a_{22}(t)x_2(t)+\ldots+a_{2n}(t)x_n(t)+f_2(t)\\ &\vdots\\ \dfrac{dx_n}{dt}&=a_{n1}(t)x_1(t)+a_{n2}(t)x_2(t)+\ldots+a_{nn}(t)x_n(t)+f_n(t). \end{aligned}\right. \Leftrightarrow \color{red}{\dot X(t)=A(t)\cdot X(t)+F(t)}. \] con condiciones iniciales para cada una de las funciones: \[ \left\{\begin{aligned} x_1(0)&=x_1\\ x_2(0)&=x_2\\ &\vdots\\ x_n(0)&=x_n\\ \end{aligned}\right. \]

Acá nos enfocarmos en sistemas lineales con coeficientes constantes, es decir, \(a_{ij}(t)=a_{ij}\in\mathbb R\).

Transformada de Laplace para resolver sistemas de EDOs lineales

La transformada de Laplace también es útil para resolver sistemas de EDOs lineales.

Use la transformada de Laplace para resolver el sistema de ecuaciones diferenciales \[\left\{ \begin{aligned} \dot x &=-x+y,\\ \dot y &=2x,\\ x(0)=0&,y(0)=1. \end{aligned} \color{red}{\Leftrightarrow \begin{pmatrix} \dot x(t)\\ \dot y(t) \end{pmatrix} = \begin{pmatrix} -1&1\\ 2&0 \end{pmatrix}\cdot \begin{pmatrix} x(t)\\ y(t) \end{pmatrix} } \right.\]

Aplicamos la transformada a ambas ecuaciones diferenciales obtenemos \[ x'=-x+y,\ x(0)=0 \Rightarrow sX(s)=-X(s)+Y(s) \quad\text{e}\quad y'=2x,\ y(0)=1 \Rightarrow sY(s)-1=2X(s). \] Es decir ahora tenemos un sistema de ecuaciones lineales \(\left\{ \begin{aligned} (1+s)X(s)-Y(s)&=0,\\ -2X(s)+sY(s)&=1,\\ \end{aligned} \right.\) que al resolver nos dice que \( X(s)=\dfrac{1}{s^2+s-2} \quad\text{e}\quad Y(s)=\dfrac{1+s}{s^2+s-2}\). Para concluir debemos usar la transformada inversa y obtenemos que \[x(t)=\mathcal L^{-1}\left( \frac{1}{s^2+s-2} \right)=\frac13e^{t}-\frac13e^{-2t},\quad y(t)=\mathcal L^{-1}\left( \frac{1+s}{s^2+s-2} \right)=\frac23e^{t}+\frac13e^{-2t}.\]

Transformada de Laplace para resolver sistemas de EDOs lineales

También se pueden resolver sistemas de segundo orden

Usar la transformada de Laplace para resolver \(\left\{ \begin{aligned} \ddot x+10x-4y&=0,\\ \ddot y-4x+4y&=0,\\ x(0)=0&,y(0)=0,\\ x'(0)=1&,y'(0)=-1. \end{aligned} \right.\)

Nuevamente aplicamos la transformada a ambas ecuaciones y obtenemos el sistema \[\left\{ \begin{aligned} (10+s^2)X(s)-4Y(s)&=1,\\ -4X(s)+(4+s^2)Y(s)&=-1,\\ \end{aligned} \right.\] que tiene como solución a \[ X(s)=\frac{s^2}{(s^2+2)(s^2+12)}=\frac{-\frac15}{s^2+2}+\frac{\frac65}{s^2+12}\quad\text{e}\quad Y(s)=-\frac{s^2+6}{(s^2+2)(s^2+12)}=\frac{-\frac25}{s^2+2}+\frac{-\frac35}{s^2+12} \] y si aplicamos la transformada inversa obtenemos que \[x(t)=-\frac{\sqrt2}{10}\sin(\sqrt 2 t)+\frac{\sqrt 3}{5}\sin(2\sqrt 3 t)\quad\text{e}\quad y(t)=-\frac{\sqrt2}{5}\sin(\sqrt 2 t)-\frac{\sqrt 3}{10}\sin(2\sqrt 3 t)\]
