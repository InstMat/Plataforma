Transformada de Laplace para resolver sistemas de EDOs lineales

La transformada de Laplace también es útil para resolver sistemas de EDOs lineales.

Use la transformada de Laplace para resolver el sistema de ecuaciones diferenciales \[\left\{ \begin{aligned} x'&=-x+y,\\ y'&=2x,\\ x(0)=0&,y(0)=1. \end{aligned} \right.\]

Si denotamos \(X(s)=\mathcal L(x(t))\) e \(Y(s)=\mathcal L(y(t))\) y aplicamos la transformada a ambas ecuaciones diferenciales obtenemos \[ x'=-x+y,\ x(0)=0 \Rightarrow sX(s)=-X(s)+Y(s) \quad\text{e}\quad y'=2x,\ y(0)=1 \Rightarrow sY(s)-1=2X(s). \] Es decir ahora tenemos un sistema de ecuaciones lineales \[\left\{ \begin{aligned} (1+s)X(s)-Y(s)&=0,\\ -2X(s)+sY(s)&=1,\\ \end{aligned} \right.\] que al resolver nos dice que \( X(s)=\dfrac{1}{s^2+s-2} \quad\text{e}\quad Y(s)=\dfrac{1+s}{s^2+s-2}\). Para concluir debemos usar la transformada inversa para obtener \(x(t)\) e \(y(t)\). Notar que \(\dfrac{1}{s^2+s-2}=\dfrac{1}{(s+2)(s-1)}=\dfrac{\frac13}{s-1}-\dfrac{\frac13}{s+2},\) de donde \[x(t)=\mathcal L^{-1}\left( \frac{1}{s^2+s-2} \right)=\frac13\mathcal L^{-1}\left( \frac{1}{s-1} \right)-\frac13\mathcal L^{-1}\left(\frac{1}{s+2}\right)=\frac13e^{t}-\frac13e^{-2t}.\]

Por otra parte \(\dfrac{1+s}{s^2+s-2}=\dfrac{1+s}{(s+2)(s-1)}=\dfrac{\frac23}{s-1}+\dfrac{\frac13}{s+2}\) de donde \[y(t)=\mathcal L^{-1}\left( \frac{1+s}{s^2+s-2} \right)=\frac23\mathcal L^{-1}\left( \frac{1}{s-1} \right)+\frac13\mathcal L^{-1}pt{\frac{1}{s+2}}=\frac23e^{t}+\frac13e^{-2t}.\]

Transformada de Laplace para resolver sistemas de EDOs lineales

Usar la transformada de Laplace para resolver \[\left\{ \begin{aligned} x''+10x-4y&=0,\\ y''-4x+4y&=0,\\ x(0)=0&,y(0)=0,\\ x'(0)=1&,y'(0)=-1. \end{aligned} \right.\]

Nuevamente aplicamos la transformada a ambas ecuaciones y obtenemos el sistema \[\left\{ \begin{aligned} (10+s^2)X(s)-4Y(s)&=1,\\ -4X(s)+(4+s^2)Y(s)&=-1,\\ \end{aligned} \right.\] que tiene como solución a \[\begin{gathered} X(s)=\frac{s^2}{(s^2+2)(s^2+12)}=\frac{-\frac15}{s^2+2}+\frac{\frac65}{s^2+12}\\ Y(s)=-\frac{s^2+6}{(s^2+2)(s^2+12)}=\frac{-\frac25}{s^2+2}+\frac{-\frac35}{s^2+12} \end{gathered}\] y si aplicamos la transformada inversa obtenemos que \[x(t)=-\frac{\sqrt2}{10}\sin(\sqrt 2 t)+\frac{\sqrt 3}{5}\sin(2\sqrt 3 t)\quad\text{e}\quad y(t)=-\frac{\sqrt2}{5}\sin(\sqrt 2 t)-\frac{\sqrt 3}{10}\sin(2\sqrt 3 t)\]
