Transformada de Laplace para resolver PVIs

Como mostramos en la sección anterior, se puede utilizar la transformada de Laplace para resolver PVIs lineales a coeficientes constantes \[\begin{aligned} a_ny^{(n)}(t)+a_{n-1}y^{(n-1)}(t)+&\ldots+a_1y^1(t)+a_0y(t)=f(t)\\ y(0)&=y_0\\ y'(0)&=y_1\\ \vdots\\ y^{(n-1)}(0)&=y_{n-1} \end{aligned}\] la estrategia es

  • Aplicar la transformada de Laplace a la ecuación, utilizando las condiciones iniciales.

  • Resolver la ecuación algebraica para \(Y(s)\).

  • Aplicar la transformada inversa para obtener \(y(t)\)

Transformada de Laplace para resolver PVIs

Resolver el PVI \[y'+3y=0,\quad y(0)=-2.\]

Usando lo aprendido en la Unidad I sabemos que la solución es \(y(t)=-2e^{-3t}\). Veamos que se obtiene lo mismo usando Transformada de Laplace.

Aplicamos la transformada de Laplace a ambos lados de la ecuación y se obtiene que \[ \mathcal L(y'+3y)=\mathcal L(0)\Rightarrow sY(s)-y(0)+3Y(s)=0\Rightarrow (s+3)Y(s)=-2 \Rightarrow Y(s)=\dfrac{-2}{s+3}, \] por lo tanto \[y(t)=\mathcal L^{-1}\left(-2\dfrac{1}{s+3}\right)=-2e^{-3t}.\]

Transformada de Laplace para resolver PVIs

Resolver el PVI \[y''-y'-2y=0,\quad y(0)=1,~y'(0)=0\]

Usando lo aprendido en la Unidad II sabemos que la solución es \(y(t)=\dfrac13 e^{2t}+\dfrac23e^{-t}\). Veamos que se obtiene lo mismo usando Transformada de Laplace.

Aplicamos la transformada de Laplace a ambos lados de la ecuación y se obtiene que \[ \mathcal L(y''-y'-2y)=\mathcal L(0)\Rightarrow s^2Y(s)-sy(0)+y'(0)-(sY(s)-y(0))-2Y(s)=0\Rightarrow Y(s)=\dfrac{s-1}{s^2-s-2}, \] pero \[ \frac{s-1}{s^2-s-2}=\frac{s-1}{(s+1)(s-2)}=\frac13\frac{1}{s-2}+\frac23\frac{1}{s+1} \] por lo tanto \[y(t)=\mathcal L^{-1}\left(\frac13\frac{1}{s-2}+\frac23\frac{1}{s+1}\right)=\frac13 e^{2t}+\frac23 e^{-t}.\]

Transformada de Laplace para resolver PVIs

Resolver el PVI \[y'+3y=13\sin(2t),\quad y(0)=6.\]

Aplicamos la transformada de Laplace a ambos lados \[\mathcal L(y'+3y)=\mathcal L(13\sin(2t)),\] de donde \[sY(s)-6+3Y(s)=13\frac{2}{s^2+4}\Rightarrow Y(s)=\frac{6+13\frac{2}{s^2+4}}{s+3}=\frac{6s^2+50}{(s+3)(s^2+4)},\] usamos fracciones parciales para obtener \[\frac{6s^2+50}{(s+3)(s^2+4)}=\frac{A}{s+3}+\frac{Bs+C}{s^2+4}=\frac{8}{s+3}+\frac{6}{s^2+4}-2\frac{s}{s^2+4},\] y así la solución del PVI es \[y(t)=\mathcal L^{-1}\left( \frac{8}{s+3}+3\frac{2}{s^2+2^2}-2\frac{s}{s^2+2^2} \right)=8e^{-3t}+3\sin(2t)-2\cos(2t).\]

Transformada de Laplace para resolver PVIs

Resolver el PVI \[y''-3y'+2y=e^{-4t},\quad y(0)=1,~ y'(0)=5.\]

Aplicamos la transformada de Laplace a ambos lados y obtenemos \[s^2Y(s)-sy(0)-y'(0)-3(sY(s)-y(0))+2Y(s)=\frac{1}{s+4}\] de donde \[Y(s)=\frac{s^2+6s+9}{(s-1)(s-2)(s+4)}\] y como vimos antes \[y(t)=\mathcal L^{-1}\left( \frac{s^2+6s+9}{(s-1)(s-2)(s+4)} \right)=-\frac{16}5e^t+\frac{25}{6}e^{2t}+\frac{1}{30}e^{-4t}.\]

Transformada de Laplace para resolver PVIs

Resolver el PVI \[y''-2y'+5y=-8e^{-t},\quad y(0)=2,~ y'(0)=12.\]

Aplicamos transformada a la ecuación y obtenemos que \[ s^2Y(s)-sy(0)-y'(0)-2(sY(s)-y(0))+5Y(s)=-\frac8{s+1} \Rightarrow Y(s)=Y(s)=\dfrac{2s^2+10s}{(s+1)((s-1)^2+2^2)}, \] y además \[\frac{2s^2+10s}{(s+1)(s^2-2s+5)}=\frac{A}{s+1}+\frac{B(s-1)+C}{(s-1)^2+2^2}=\frac{-1}{s+1}+\frac{3(s-1)+8}{(s-1)^2+2^2},\] por lo tanto obtenemos que \[y(t)=\mathcal L^{-1}\left(\frac{-1}{s+1}+3\frac{s-1}{(s-1)^2+2^2}+4\frac{2}{(s-1)^2+2^2}\right)=-e^{-t}+3e^{t}\cos(2t)+4e^{t}\sin(2t).\]

Ejercicios

Use la transformada de Laplace para resolver los problemas de valores iniciales.

  1. \(y'-y=1\), \(y(0)=0\).
  2. \(y'+6y=e^{4t}\), \(y(0)=2\).
  3. \(y''+5y'+4y=0\), \(y(0)=1\), \(y'(0)=0\).
  4. \(y''-y'=e^{4t}\cos t\), \(y(0)=0\), \(y'(0)=0\).
  5. \(y''-4y+4y=t^3\), \(y(0)=1\), \(y'(0)=0\).
  6. \(2y'''+3y''-3y'-2y=e^{-t}\), \(y(0)=0\), \(y'(0)=0\), \(y''(0)=1\).
  7. \(y^{(4)}-4y=0\), \(y(0)=1\), \(y'(0)=1\), \(y''(0)=-2, y'''(0)=0\).
