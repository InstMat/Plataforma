Ecuaciones diferenciales ordinarias

Ingeniería Civil

Ecuaciones diferenciales con funciones homogéneas

Una función se dice homogénea de grado \(\alpha\) si se cumple que \[f(tx,ty)=t^\alpha f(x,y)\quad\text{ para todo }t,x,y.\]

  1. La función \(f(x,y)=x+y\) es homogénea de grado \(1\) pues \[ f(tx,ty)=tx+ty=t(x+y)=tf(x,y) \]
  2. La función \(f(x,y)=x^2-y^2+xy\) es homogénea de grado \(2\) pues \[ f(tx,ty)=(tx)^2-(ty)^2+(tx)\cdot (ty)=t^2x^2-t^2y^2+t^2xy=t^2(x^2-y^2+xy)=t^2f(x,y) \]

Ecuaciones diferenciales con funciones homogéneas

Una ecuación diferencial de primer orden en la forma \[M(x,y)dx+N(x,y)dy=0\] es homogénea si las funciones \(M,~N\) son homogéneas del mismo grado. Análogamente, una ecuación de la forma \[y'=f(x,y)\] es homogéneas si la función \(f\) es homogénea de grado \(\color{red}{0}\).

Para resolver una ecuación homogénea se considera uno de los siguientes cambios de variables \[y=xv \text{ o bien } y=\dfrac{x}{u}.\] La ecuación resultante debería ser una ecuación diferencial separable para la función \(v\) o la función \(u\) según sea el caso.

Ecuaciones diferenciales con funciones homogéneas

Resolver la ecuación \((x+y)\mathrm dx+x\mathrm dy=0\).

Notar que \(M(x,y)=x+y\) y \(N(x,y)=x\) son homogéneas de grado \(1\). Si consideramos la transformación \(y=xv\) y usamos la regla del producto obtenemos que \(\mathrm{d}y=x\mathrm{d} v+v\mathrm{d} x\) de donde la ecuación se puede reescribir como \[ \begin{align*} (x+xv)\mathrm{d} x+x(x\mathrm{d} v+v\mathrm{d} x)&=0\\ (x+2xv)\mathrm{d} x+x^2\mathrm{d} v&=0\\ x(1+2v)\mathrm{d} x+x^2\mathrm{d} v&=0\\ \frac1x\mathrm{d} x+\dfrac{1}{1+2v}\mathrm{d} v&=0\\ \int \frac1x\mathrm{d} x+\int \dfrac{1}{1+2v}\mathrm{d} v&=0\\ \ln|x|+\frac12\ln|1+2v|&=C, \end{align*} \] de donde volviendo a la variable original obtenemos que \[ \ln|x|+\frac12\ln\left|\frac{x+2y}{x}\right|=C. \]

Ecuaciones diferenciales con funciones homogéneas

Resolver la ecuación \(y'=\dfrac{x+3y}{3x+y}\). Ayuda: Use la transformación \(y=\dfrac{x}{u}\) y la descomposición en fracciones parciales \(\dfrac{3u+1}{u(1-u)(1+u)} = \dfrac{1}{u}+\dfrac{1}{1+u}+\dfrac{2}{1-u}\).

Notar que en la función \(f(x,y)\) es homogénea de grado \(0\). Si hacemos la transformación \(y=\dfrac{x}{u}\) obtenemos que la ecuación queda \[ \frac{1}{u}-\frac{xu'}{u^2}=y'=\dfrac{x+3y}{3x+y}=\dfrac{y(\frac{x}y+3)}{y(3\frac{x}y+1)}=\dfrac{u+3}{3u+1}. \Rightarrow xu'=u^2\left({\frac1u-\dfrac{u+3}{3u+1}}\right)=\dfrac{u(1-u)(1+u)}{3u+1} \] es decir \[ \int\dfrac{3u+1}{u(1-u)(1+u)}\mathrm{d} u = \int\frac{1}{x}\mathrm{d} x \Rightarrow \ln|u|+\ln|1+u|-2\ln|1-u|=\ln|x|+C. \] Por lo tanto la solución satisface \[ \ln\left|\frac{x}{y}\right|+\ln\left|1+\frac{x}{y}\right|-2\ln\left|1-\frac{x}{y}\right|=\ln|x|+C \]

Ecuaciones diferenciales con funciones homogéneas

Resuelva la ecuación diferencial \((x^2+y^2)\mathrm dx+(x^2-xy)\mathrm dy=0\). Ayuda: Use la transformación: \(y=xv\).

Ecuación diferencial de Bernoulli

Una ecuación diferencial se dice que es de Bernoulli de grado \(n\) si se puede escribir en la forma \[y'+p(x)y=f(x)y^{n}, \text{ donde } n\neq 0 \text{ y } n\neq 1.\]

Para resolver una ecuación diferencial de Bernoulli se considera el siguiente cambio de variable \[u=y^{1-n}.\] La ecuación resultante debería ser una ecuación diferencial lineal para la función u.

Ecuación diferencial de Bernoulli

Resolver \(xy'+y=x^2y^2\).

Esta es una ecuación de Bernoulli de grado \(2\), por lo que debemos hacer la transformación \(u=y^{1-2}\Rightarrow y=\dfrac{1}{u}\), lo que implica \(y'=-\dfrac{1}{u^2}u'\), por lo tanto \[ xy'+y=-\frac{x}{u^2}u'+\frac{1}{u}=x^2\left(\frac1{u}\right)^2, \] y al multiplicar por \(u^2\) se obtiene la ecuación lineal \[ -xu'+u=x^2\Leftrightarrow xu'-u=-x^2. \] Para resolver esta ecuación la llevamos a la forma canónica \(u'-\dfrac{1}{x}u=-x\) y usamos el factor integrante \[ e^{\int -\frac1{x}\mathrm{d} x}=e^{-\ln|x|}=\frac{1}{|x|}\rightarrow \mu(x)=\frac{1}x \] por lo tanto \[ \frac{1}{x}u(x)=\int -x\cdot \dfrac{1}{x}\mathrm{d} x=-x+C\Rightarrow y(x)=\frac{1}{-x^2+Cx}. \]

Ecuación diferencial de Bernoulli

Resolver \(y'+\left({4x-\dfrac1{x}}\right)y=-2y^2\).

Esta es una ecuación de Bernoulli de grado \(2\), por lo que debemos hacer la transformación \(u=y^{1-2}\Rightarrow y=u^{-1}\), lo que implica \(y'=-\dfrac{u'}{u^2}\), por lo tanto \[ y'+\left({4x-\dfrac1{x}}\right)y=-\dfrac{u'}{u^2}+\left({4x-\dfrac1{x}}\right)\frac{1}{u}=-2\left(\frac1{u}\right)^2, \] y al multiplicar por \(u^2\) se obtiene la ecuación lineal \[ -u'+\left({4x-\dfrac1{x}}\right)u=-2\Leftrightarrow u'-\left({4x-\dfrac1{x}}\right)u=2. \] Para resolver usamos el factor integrante \( e^{\int -\left({4x-\frac1{x}}\right)\mathrm{d} x}=e^{-2x^2+\ln|x|}=|x|e^{-2x^2}\rightarrow \mu(x)=xe^{-2x^2} \) por lo tanto \[ xe^{-2x^2}u(x)=\int 2xe^{-2x^2}\mathrm{d} x=-\frac12e^{-2x^2}+\tilde C \Rightarrow u(x)=-\frac{1}{2x}+\frac{\tilde Ce^{2x^2}}{x} \Rightarrow y(x)=\frac{2x}{\color{red}{C} e^{2x^2}-1} . \]

Ecuación diferencial de Ricatti

Una ecuación diferencial se dice que es de Ricatti si se puede escribir en la forma \[y'=p(x)+q(x)y+r(x)y^2.\]

Para resolver una ecuación diferencial de Ricatti es necesario conocer de antemano una solución particular \(y_1\). Si se conoce dicha solución particular se puede obtener una familia de soluciones haciendo la transformación \[u=y-y_1.\]

La ecuación resultante debería ser una ecuación diferencial de Bernoulli para la función \(u\) la que se puede resolver con el método anteriormente visto.

Ecuación diferencial de Ricatti

Resolver la ecuación diferencial \(y'=2x^2+\dfrac{y}x-2y^2\) sabiendo que \(y_1(x)=x\) es una solución.

Primero notamos que efectivamente \(y_1(x)=x\) es solución, puesto que \(y_1'=1\) y \(2x^2+\frac{y_1}x-2y_1^2=2x^2+1-2x^2=1\). Con esto, consideramos \(u=y-y_1=y-x\) y obtenemos \[ \begin{align*} u'&=y'-1\\ &=2x^2+\frac{y}x-2y^2-1\\ &=2x^2+\frac{u+x}x-2(u+x)^2-1\\ &=2x^2+\frac{u}x+1-2(u^2+2xu+x^2)-1\\ &=2x^2+\frac{1}{x}u-2u^2-4xu-2x^2\\ &=\left({\frac{1}{x}-4x}\right)u-2u^2, \end{align*} \] de donde \(u'+\left({4x-\frac1{x}}\right)u=-2u^2\), que es la ecuación de Bernoulli que resolvimos anteriormente, por lo tanto, como \(y=u+x\) se sigue que \[ u(x)=\frac{2x}{Ce^{2x^2}-1}\Rightarrow y(x)=x+\frac{2x}{Ce^{2x^2}-1}. \]

Ejercicios

1. Resuelva las siguientes EDOs homogéneas.

  1. \(\dfrac{dy}{dx} = \dfrac{x^2+y^2}{2x^2}\)
  2. \(\dfrac{dy}{dx} = \dfrac{x-y}{x+y} \)
  3. \(\dfrac{dy}{dx}=\dfrac{3y-4x}{x}\)

2. Resuelva las siguientes EDOs de Bernoulli.

  1. \(y' + \dfrac{4}{x}y = {x^3}{y^2}\)
  2. \(y' = 5y + e^{- 2x}y^{-2}\)
  3. \(y' + \dfrac{y}{x} - \sqrt y = 0\)

3. Resuelva las siguientes EDOs de Ricatti.

  1. \(y' = y + {y^2} + 1\). Ayuda: Esta EDO es separable.
  2. \(y' + {y^2} = \dfrac{2}{{{x^2}}}\) usando la solución particular \(y_1(x)=\dfrac{2}{x}\).
  3. \({x^3}y' + {x^2}y - {y^2} = 2{x^4}\) usando la solución particular \(y_1(x)=x^2\).
