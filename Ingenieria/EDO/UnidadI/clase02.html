Separación de variables

Este método aplica para ecuaciones de la forma \[y'=f(x)g(y),\] donde \(f(x)\) y \(g(y)\) son funciones conocidas.

Las siguientes E.D.O.s son separables

  1. \(y'=x\).
  2. \(y'=x\cdot y\).
  3. \(y'=\dfrac{y^2+1}{e^x}\).
  4. \(y'=e^{x+y}\).

Las siguientes E.D.O.s no son separables

  1. \(y'=x+y\).
  2. \(y'=\dfrac{x}{x+y}\).
  3. \(y'=\ln(x-y)\).
  4. \(y'=y^x\).

Separación de variables

Para resolver una E.D.O. separable la idea es “separar” variables: \[y'=f(x)g(y) \Leftrightarrow \frac{dy}{dx}=f(x)g(y) \Rightarrow \frac{1}{g(y)}\mathrm dy=f(x)\mathrm dx\] de donde podemos integrar para obtener \[\int \frac{1}{g(y)}\mathrm dy=\int f(x)\mathrm dx\] Luego si denotamos \(\displaystyle G(y)=\int \frac{1}{g(y)}dy\) y \(\displaystyle F(x)=\int f(x)dx\) a las respectivas primitivas, obtenemos la solución implícita \[G(y)=F(x)+C.\]

  • Notar que lo el método entrega una familia de soluciones implícitas ya que para cada valor de la constante \(C\) se obtienen soluciones implícitas distintas.
  • Si se conoce la función inversa de \(G\) entonces es posible despejar \(y\) en función de \(x\) para obtener la solución explícita: \[y(x)=G^{-1}(F(x)+C).\]
  • Si se entrega una condición inicial, entonces se puede determinar el valor de la constante \(C\).

Separación de variables

Resolver \(y'=-\dfrac xy\).

Escribimos \(y'=-\dfrac xy\Leftrightarrow y\mathrm dy=-x\mathrm dx\) y luego integramos a ambos lados para obtener \[\int y\mathrm dy=-\int x\mathrm dx\Leftrightarrow y^2=-x^2+C,\] es decir, obtenemos la solución implícita \[x^2+y^2=C.\]

Separación de variables

Resolver la ecuación \(xy'=y\).

Observar que si existe \(y_0\) tal que \(g(y_0)=0\) entonces la ecuación \(y'=f(x)g(y)\) también tiene como solución a la función constante \(y(x)=y_0\).

Cuando esto ocurre diremos que la E.D.O. tiene una solución constante. No olvidar la búsqueda de soluciones constantes.
