# Cómo agregar un módulo

1. Edita el archivo `data/carreras.json` y agrega el nuevo módulo en la sección correspondiente.
2. Crea el contenido del módulo en la carpeta de la carrera/asignatura.
3. Si es una presentación, convierte el archivo `.tex` a HTML usando Pandoc.
4. Actualiza el archivo `lecciones.json` si corresponde.
5. Verifica que el módulo aparezca en el menú lateral y se pueda acceder desde la interfaz.

**Tip:** Revisa el archivo `.github/copilot-instructions.md` para detalles técnicos.
